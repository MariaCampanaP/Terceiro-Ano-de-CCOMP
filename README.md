# ☕︎ Terceiro Ano de CCOMP

Repositório contendo materiais e exercícios das disciplinas do **terceiro ano** do curso de **Ciência da Computação** na UNESPAR.

## ☕︎ Sobre o Repositório

Este repositório reúne o conteúdo estudado ao longo do segundo ano, organizado por disciplina:

- *Banco de Dados*
- *Sistemas Operacionais*
- *Sistemas Microcontrolados*
- *Inteligência Artificial*

## ☕︎ Objetivo

Centralizar os códigos, exercícios e anotações produzidos durante o segundo ano, servindo como material de estudo, revisão e portfólio acadêmico.

## ☕︎ Tecnologias e Linguagens Utilizadas

- SGBD relacional (MySQL, PostgreSQL ou similar);
- Python;
- C;
- C++;
- Oracle VirtualBox;
- Ubuntu (64-bit);
- Code::Blocks;
- Arduino IDE;
- Tinkercad;
- Draw.io.
  
## ☕︎ Como Usar

## Como clonar este repositório

```bash
git clone https://github.com/MariaCampanaP/Terceiro-Ano-de-CCOMP.git
```

Isso vai baixar todas as disciplinas do 3° ano.
Se quiser baixar apenas a pasta de uma disciplina específica, veja as instruções de sparse-checkout no README de cada pasta.
