# ☕︎ Sistemas Microcontrolados

Repositório contendo exercícios da disciplina **Sistemas Microcontrolados** do terceiro ano de **Ciência da Computação** na UNESPAR.

## ☕︎ Sobre o Repositório

Este repositório reúne implementações de conceitos fundamentais de **Sistemas Microcontrolados**, contendo códigos em C++ adaptado para programação de placas como o Arduino IDE, além da vista do circuito e da vista esquemática.

- *Segundo Projeto - Led Alternada*
- *Terceiro Projeto - Efeito "Carregando"*
- *Quarto Projeto - Número Binário*
- *Quinto Projeto - Aprendendo a usar botão*
- *Sexto Projeto - Inverter led*
- *Sétimo Projeto - Pressionando Botão*
- *Oitavo Projeto - Apagar e Acender*
- *Nono Projeto - Potenciômetro*
- *Décimo Projeto - Contador*
- *Décimo Primeiro Projeto - Servo Motor*
- *Décimo Segundo Projeto - Cancela de Semáforo*
- *Décimo Terceiro Projeto - Ré de Carro*
- *Décimo Quarto Projeto - Portão Eletrônico*
- *Décimo Quinto Projeto - Pulse Width Modulation*
- *Décimo Sexto Projeto - Potenciomêtro Controlando Led*

- *Décimo Oitavo Projeto - Controle de Brilho Inverso*

# ☕︎ Objetivo

Os códigos, as vistas do circuito e as representações esquemáticas servem como material de estudo e prática para reforçar os conceitos abordados na disciplina.

## ☕︎ Tecnologia e Linguagens Utilizadas

- C++;
- Arduino IDE;
- Tinkercad.

## Como clonar apenas esta pasta

Este repositório contém todas as disciplinas do 3° ano. Se quiser baixar **somente** a pasta "Sistemas Microcontrolados", use o sparse-checkout do Git:

```bash
git clone --no-checkout https://github.com/MariaCampanaP/Terceiro-Ano-de-CCOMP.git
cd Terceiro-Ano-de-CCOMP
git sparse-checkout init --cone
git sparse-checkout set "Sistemas Microcontrolados"
git checkout main
```
